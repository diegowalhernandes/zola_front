# Bellora — Frontend

Marketplace de profissionais de beleza e estética.

## Scripts

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
```

## Variáveis

Copie `.env.example` para `.env` e configure `VITE_API_URL` se necessário.

## Marca

- **Slogan:** Sua beleza, seu momento.
- Tokens: `src/design/brand.ts`
- Design system: `DESIGN_SYSTEM.md`
